# UI / UX Standards

## 1. Product Feel

The product should feel:

- premium
- calm
- modern
- motivating
- healthy
- friendly
- mobile-first
- frictionless

## 2. Onboarding Rules

Recommended steps:

1. welcome
2. goal
3. age / height / weight
4. activity level
5. target pace
6. calculated target
7. account creation
8. first action
